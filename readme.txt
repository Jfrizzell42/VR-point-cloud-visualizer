Requirements:
Ubuntu 16.04
Unreal engine 4.18.3
ROS kinetic
ROS turtlebot package
To run
Download unreal project at: https://drive.google.com/open?id=1P7RQvqY1KwErL-6PFKLu8XRWI00Fqm4G
run project in editor
open ros and run a roscore
make and run move.py
utilize keyboard controls
(may be difficult to run since you will need the linux distrobution of unreal on the same machine as ROS)
